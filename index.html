YTY Studio

Youth Teaching Youth

YTY Studio 由一群專業電視主持及司儀於2018年創辦，YTY 除了是創辦人名字的縮寫，同時也代表著Youth Teaching Youth，學習需要保持初心，保持心境年輕，才能邁步向前。  

YTY Studio提供一系列與說話技巧相關的私人及企業培訓課程，包括主持及司儀技巧、演說技巧、職場溝通技巧、營銷技巧、CEO說話魅力、擴闊客源技巧等，並籌辦與說話技巧相關的比賽及活動。

YTY Studio 亦為不同企業、品牌、政府部門、及各大機構提供專業司儀服務，包括頒獎典禮、週年晚宴、新聞發布會、產品發布會、論壇及品牌活動等。

企業培訓

YTY Studio 為各大企業提供演說技巧、銷售技巧、與客戶溝通技巧、上司下屬溝通技巧、擴闊客源、處理投訴等培訓課程，分為線上及線下面對面形式，致力提升在職人士的說話及表達技巧，從而提高業績及工作效率。過往服務對象包括：AIA人壽保險、煤氣公司、協成行、好師傅、i-Rent、深圳彩攝會等。


學校、機構及協會的培訓及分享

YTY Studio 曾為不同小學、中學、大學、機構及協會，提供一系列說話技巧相關培訓課程，分為線上及線下形式，包括司儀、演說及溝通技巧等。導師們亦經常擔任分享嘉賓，分享工作經驗及說話技巧，激勵觀眾士氣。過往服務對象包括：香港大學、香港城市大學、香港理工大學、上水鳳溪第一中學、JCI 、扶輪社、香港建造業議會、香港貿發局、保險業協會、傷殘青年協會、深圳讀書會等。




司儀及演講技巧班

YTY Studio 提供不同程度的司儀及演講技巧班，提升學員的主持及演講能力，我們的課程特色如下：

• 課堂上，理論及實踐並重，導師會引導學員解決活動上遇到的不同情況及突發事情。
• 導師會挑選有潛質的學員，推薦給客戶，並提供真正當活動司儀的機會。
• 所有課程均為小班教學，讓學員有充足的實踐機會。
• 完成課程後，YTY Studio 會頒發證書予學員，以茲證明。



活動司儀服務

YTY Studio 為客戶提供專業三語司儀服務，所有主持均具有豐富電視主持及活動司儀經驗，確保活動順利進行。部份主持更精通韓語和日語，可迎合不同客戶需要。


客戶

我們曾經與不同客戶合作，包括知名企業、時尚品牌、政府部門、非牟利機構、知名學府等，提供司儀服務、培訓課程等。
